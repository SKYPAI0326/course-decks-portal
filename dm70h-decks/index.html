操作方式 · How to lecture

Navigation · 翻頁

四種方式都行

← → 鍵盤翻頁、滑鼠滾輪、觸控板左右滑動、底部圓點直接跳。

Overview · 索引

ESC 看全貌

按 Esc 進入縮圖總覽，看清整份簡報的章節節奏，再點任一頁跳到該頁。再按 Esc 退出。

Pipeline · 互動

逐步點亮

有些頁面是 pipeline 流程（如 GTM 七步、JTBD 五問）。按 → 會逐個點亮 step，全亮才翻下一頁。可控制講課節奏。

Aspect · 比例

16:9 全螢幕投影

所有 deck 設計給 16:9 全螢幕投影。投影前按 F11（Mac 用 Safari/Chrome 全螢幕鍵）達到最佳效果。

Theme · 主題色

牛皮紙底

暖米黃底 + 深棕字（Kraft Paper）。長時間講課對眼睛友善，比白底投影更耐看。

Browser · 相容

現代瀏覽器

Chrome / Edge / Safari / Firefox 最新版皆可。不依賴後端，單檔離線也能跑（含 WebGL 背景與動效）。

About · 關於本簡報集

一張完整的數位行銷地圖

翻開這套簡報，你會看到 70 小時數位行銷課程的全部教學內容：從消費者心理學打底，一路走過生成式 AI、市場洞察、社群、SEO、GA4、GTM、廣告、Email，最後收束在整合實戰專案。

十個模組互相回扣——M1 的人性洞察會在 M4 注意力心理學、M8 廣告文案再次出現；M2 的 AI 工具箱解鎖後續所有模組的 AI 接口。可以順著走完整 70 小時，也可以單獨打開某一份補強單一主題；每份簡報都用「概念 → 案例 → 練習」三段式編排，幕間留 hero 頁讓眼睛呼吸。

視覺基調借自 Monocle 雜誌——牛皮紙底、衬線大標題、極簡留白，把注意力留給內容本身。長時間讀比白底耐看。
